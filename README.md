# Estrategia_de-_Expansi-n_2026
Este proyecto utiliza Data Science para determinar la ubicación del primer local físico de una empresa. Analizamos más de 2.000 ventas y conectamos con la API de OpenWeather para optimizar la logística.

Objetivos
Identificar el barrio con mayor Ticket Promedio.

Validar la hipótesis del local como Hub Logístico en días de lluvia.

Automatizar la obtención de datos climáticos mediante servicios de Google Cloud.

Principales Hallazgos
Barrio Ganador: Caballito ($86.048 por ticket).

Eficiencia: El local físico reduce costos de envío al cubrir zonas de alta demanda online.

Herramientas

Python (Pandas, Seaborn)

API REST (OpenWeatherMap)

Google Colab Secrets
